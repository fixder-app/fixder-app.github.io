Fixder is an app development tool for Chez Scheme users. The features are:


One Code One Executable The app built by Fixder can be only one executable file. This is really awsome.

One Code All Distros Fixder can pack the app in various formats for the distros, like RPM, DEB etc.

Bootstraped By Itself Fixder is built by itself and coded in Chez Scheme totally.

Compiled Code For Privacy Fixder compiles code before making packages. So your apps can be quick and private.
